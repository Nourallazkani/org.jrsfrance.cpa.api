Les principes de base du language :

- primitifs et type objets correspondant (int et Integer, double et Double, etc...)
- les blocs (for, if, while, switch)
- la classe Object, dont toutes les classes héritent.
- les niveaux de visibilités (private, public, protected)
- la gestion des exceptions (throw, throws, try, catch, finally, le try with resources apparu en java 7)
- la classe String et son fonctionnement particulier (le pool de string, le fait que String soit immutable, l'opérateur +-), les StringBuffer
- la manipulation des data (java.util.Date)
- les collections (java.util.List, java.util.Set), les maps (java.util.Map)

Les principes objet : 
- classe, interface, classe abstraite
- héritage
- encapsulation
- polymorphisme

depuis java 5 :
- les enums
- les varargs
- les types génériques
- les annotations

depuis java 8 :
- les expressions lambda
- les streams
- les Optional
